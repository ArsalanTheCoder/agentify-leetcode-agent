## 🌟 Project Overview

**Agentify** is a cutting-edge web application designed to empower university students and junior engineers in their coding journey. Leveraging advanced AI, Agentify provides personalized daily challenges, curated premium content, and a competitive environment to help users master algorithms, excel in technical interviews (especially for FAANG companies), and track their continuous progress.

This project is built with Next.js using the App Router, JavaScript, Tailwind CSS, and other modern web technologies to ensure a scalable, high-performance, and delightful user experience.

## ✨ Core Features (What We're Building)

Agentify aims to provide a comprehensive platform with the following key features:

* **AI-Powered Daily Challenges:** Personalized coding problems delivered daily, tailored to the user's skill level based on GitHub activity.
* **Premium Problem Playlists:** Curated, company-specific problem sets (e.g., "Meta Top 50," "Google Interview Prep").
* **Expert Video Library:** A collection of video roadmaps and problem-solving sessions by industry experts.
* **Contest & Quiz Hub:** Opportunities to participate in weekly coding contests, topic-specific quizzes, and special competitions.
* **Dynamic Dashboard:** A personalized overview of progress, streaks, skill analytics, and upcoming activities.
* **User Profiles & Achievements:** Comprehensive user profiles, skill blueprints, and digital certificates for achievements.
* **Seamless GitHub Integration:** For code analysis and progress tracking.

## 🚀 Getting Started

Follow these instructions to get your local development environment up and running.

### Prerequisites

* Node.js (LTS version recommended)
* npm (comes with Node.js)
* Git

### Installation

1.  **Clone the Repository:**
    If you're a new developer joining the team, or setting up on a new machine, clone the project from GitHub:

    ```bash
    git clone [https://github.com/ArsalanTheCoder/agentify-leetcode-agent.git](https://github.com/ArsalanTheCoder/agentify-leetcode-agent.git)
    ```

2.  **Navigate into the Project Directory:**

    ```bash
    cd agentify-leetcode-agent
    ```

3.  **Install Dependencies:**

    ```bash
    npm install
    ```

4.  **Run the Development Server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000/) in your browser to see the application.

## 🔄 How to Pull Latest Changes (For Existing Developers)

If you're an existing team member and want to ensure your local repository is up-to-date with the latest changes from GitHub:

1.  **Navigate to your project directory:**

    ```bash
    cd agentify-leetcode-agent
    ```

2.  **Pull the latest changes:**

    ```bash
    git pull origin main
    ```

    This command will fetch and merge any new commits from the `main` branch on GitHub into your local branch. If there are any merge conflicts, Git will prompt you to resolve them.
